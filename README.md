**HR Analytics Dashboard - Workforce Attrition Analysis**

📊 **Project Overview**

This project involves the creation of a comprehensive HR Analytics Dashboard using Power BI. The goal was to transform a raw dataset of over 1,400 employees into an interactive visual tool that identifies key drivers of employee attrition and provides a clear view of workforce demographics.  

By leveraging this dashboard, HR managers can perform data-driven analysis to improve retention strategies and understand organizational health across multiple departments.

**🎯 Business Objectives**

• Identify Attrition Patterns: Analyze why employees are leaving based on department, job role, and salary slabs.  

• Demographic Tracking: Visualize the workforce age distribution and gender balance.  

• Satisfaction Correlation: Map job satisfaction levels against attrition rates to pinpoint high-risk areas.  

• Interactive Filtering: Provide a dynamic experience using slicers to drill down into specific departments like IT, Operations, Sales, and Finance.

**🛠️ Technical Skills Applied**

• Data Cleaning (Power Query): Handled a dataset with 37 columns, removing duplicates and standardizing categorical data (e.g., fixing spelling inconsistencies and handling null values).  

• DAX Measures: Developed custom formulas for key metrics, including:

    • Attrition Rate % (using the DIVIDE function to handle errors).  

    • Active Employee Count.  

    • Average Age & Experience.  

• Data Visualization: Implemented KPI cards, Doughnut charts, Bar charts, Area charts for trends, and Matrix visuals for deep-dive analysis.  

• UI/UX Design: Focused on professional dashboard aesthetics using consistent themes, rounded corners, and shadow effects for a modern look.

**🔍 Key Insights from the Dashboard**

• High-Risk Departments: Identified that the Administration department has a significantly higher attrition rate (20.8%) compared to the organizational average.  

• Salary Correlation: Discovered that a large portion of attrition occurs within the 6-10 LPA salary slab.  

• Experience Trends: Analyzed how attrition fluctuates across different experience levels, showing specific peaks at early-career stages.
